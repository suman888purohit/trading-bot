# Binance Futures Trading Bot

A robust Python trading bot for Binance Futures Testnet with comprehensive validation, logging, and error handling.

## Features

- ✅ Place MARKET and LIMIT orders on Binance Futures Testnet
- ✅ Support for both BUY and SELL orders
- ✅ Comprehensive input validation
- ✅ Detailed logging (API requests, responses, errors)
- ✅ CLI interface with interactive mode
- ✅ Proper error handling for API and network failures
- ✅ Order history tracking

## Prerequisites

- Python 3.8+
- Binance Testnet Account
- Binance Testnet API credentials

## Setup

### 1. Get Testnet API Credentials

1. Register at [Binance Futures Testnet](https://testnet.binancefuture.com/)
2. Generate API key and secret from the dashboard
3. Fund your test account with test USDT (available from faucet)
